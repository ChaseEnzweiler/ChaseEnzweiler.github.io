Bear Maps

    The Bear Maps web application is a project from UC Berkeley's CS61B course where the goal was to implement the backend of the app. Implementing the backend for this project consisted of rastering images to be served by the frontend, parsing and storing map data, finding shortest routes between locations, and implementing autocomplete when searching for specific locations.

Home Page

Here is the initial view of the map when first accessing the web application. The user has capabilities to zoom in or out and click to drag the screen to a portion of the map they are interested in.

Location Search

A user can search for a location by typing into the text box in the upper left corner of the screen. As the user types, an autocomplete feature will provide some location recommendations. In this example, the user is looking for Northside Cafe.

Location Found

When the user selects a location, a red circle with a black center will be placed at the selected location.

Route

To find a route between two locations on the map, the user can double click anywhere on the map to set a starting point and then double click again somewhere else on the map to set an ending point. A blue line will then be drawn on the map representing the shortest route between those two locations. The example takes us from Haste Street to Northside Cafe.

Navigation

Turn by turn directions are also provided for each route set by the user. For this example, directions are provided for the route from People's Park on Dwight Way to the California Memorial Stadium entrance.
